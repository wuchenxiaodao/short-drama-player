## 📊 项目亮点

### 技术栈
- 后端：Java 17 + Spring Boot 3.2 + JWT + JPA
- 前端：Next.js 14 + TypeScript + Tailwind CSS + Zustand
- 部署：Docker Compose / Kubernetes

### 项目规模
- 前端代码：50+ 文件
- 后端代码：100+ 文件
- API 接口：62 个

### 核心功能
- 沉浸式播放体验
- 多种互动类型（投票、答题、抉择、彩蛋）
- 实时弹幕系统
- 完整社交功能
- 用户积分和成就系统

## 🎤 讲解要点

### 开场
> "大家好，今天为大家演示的是短剧互动播放器。这是一款为短剧设计的沉浸式互动娱乐平台，核心体验是'观看-互动-再参与'的闭环。"

### 互动系统（重点）
> "这是项目的核心亮点——互动系统。当视频播放到特定时间点时，会弹出互动窗口。我们支持投票、答题、抉择、彩蛋四种互动类型。"

### 技术亮点
> "后端采用 Spring Boot，前端采用 Next.js 14，支持服务端渲染。数据库支持 H2 和 MySQL 双模式，部署支持 Docker 一键部署。"
